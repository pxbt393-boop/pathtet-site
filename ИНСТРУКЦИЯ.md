# ИНСТРУКЦИЯ ДЛЯ ЧАЙНИКОВ 🧸

## Как опубликовать сайт с паштетом в интернете

---

### Шаг 1. Зарегистрируйся на GitHub

1. Открой сайт **github.com**
2. Нажми **Sign up** (справа сверху)
3. Введи почту, придумай пароль, подтверди почту
4. Готово ✅

---

### Шаг 2. Создай репозиторий (хранилище для сайта)

1. Нажми на **+** в правом верхнем углу → **New repository**
2. В поле **Repository name** напиши: `pathtet-site`
3. Ниже выбери **Public** (чтобы сайт был доступен всем)
4. Нажми зелёную кнопку **Create repository**

---

### Шаг 3. Загрузи туда файлы

1. На открывшейся странице найди секцию **…or push an existing repository from the command line**
2. Увидшь три строчки. Скопируй **первую** (она начинается с `git remote add origin`):
   ```
   git remote add origin https://github.com/твой_username/pathtet-site.git
   ```
3. Открой **Терминал** (на Mac: нажми `Cmd + Пробел` → напиши `Терминал` → Enter)
4. В терминале напиши команду (вставь скопированное):
   ```
   cd /Users/anastasiaborodina/Desktop/OpenTests/Неглиже/pathtet-site
   ```
   Нажми **Enter**
5. Теперь вставь ту самую скопированную строчку, например:
   ```
   git remote add origin https://github.com/твой_username/pathtet-site.git
   ```
   Нажми **Enter**
6. Напиши:
   ```
   git push -u origin main
   ```
   Нажми **Enter**

Если попросит логин/пароль — введи свой GitHub логин и пароль.

---

### Шаг 4. Включи сайт

1. Зайди в свой репозиторий на GitHub (`github.com/твой_username/pathtet-site`)
2. Нажми на вкладку **Settings**
3. Слева в меню выбери **Pages**
4. В разделе **Branch** выбери `main`, рядом выбери `/ (root)`
5. Нажми **Save**

Через 1-3 минуты твой сайт появится по адресу:

👉 **https://твой_username.github.io/pathtet-site**

---

### Шаг 5. Если хочешь что-то поменять на сайте

1. Скажи мне — я изменю файлы
2. Потом открой Терминал и напиши по очереди:

   ```
   cd /Users/anastasiaborodina/Desktop/OpenTests/Неглиже/pathtet-site
   git add .
   git commit -m "обновление"
   git push
   ```

3. Через минутку изменения появятся на сайте
